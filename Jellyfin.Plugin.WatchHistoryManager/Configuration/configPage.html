Auto-Mark Previous Episode

If enabled, the plugin can mark the previous episode as watched when the next episode of the same series is started.
Example: 75 means the previous episode must have been watched to at least 75 percent before it can be marked as watched.
Example: 240 means the previous episode can be marked as watched if 4 minutes or less are remaining.
If enabled, episodes from season 0 are ignored by the automatic watch-state logic.

Start Point Button

If enabled, the plugin allows the Jellyfin Web helper script to show a button for setting the current episode as the new start point of a series.
